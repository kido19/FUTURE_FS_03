# Zemen Bakery & Coffee 🇪🇹🥐☕

A premium, modern frontline website built for an artisanal Ethiopian bakery and gourmet coffee shop. This project seamlessly blends deep Ethiopian heritage with luxury web aesthetics, featuring a bilingual (English & Amharic) interface, a custom "Golden Hour" dark-mode palette, and high-performance interactive elements.

## Features ✨

- **Premium UI/UX Design**: An elegant dark theme mapping a bespoke "Golden Hour" palette (featuring Deep Charcoal, Rich Gold, and Crisp White).
- **Interactive Order Modal**: A custom-built JavaScript pop-up modal allowing users to submit orders seamlessly without page redirection.
- **Bilingual Content**: Integrated Amharic and English typography prioritizing dual-language accessibility.
- **Scroll Animations**: High-performance, lightweight `IntersectionObserver` integrations for smooth reveal animations as the user scrolls.
- **Fully Responsive Structure**: Designed from the ground up using CSS Grid and Flexbox for perfect alignment across mobile, tablet, and desktop viewports.

## Technologies Used 🛠️

- **HTML5**: Semantic markup acting as the robust skeleton.
- **CSS3 (Vanilla)**: Extensive use of CSS Custom Properties (Variables), advanced pseudo-elements, `backdrop-filter` for glass-morphism effects, and highly curated `text-shadow` layering for maximum legibility over complex background images.
- **JavaScript (Vanilla)**: DOM manipulation, event listeners for the modal functionality, and Intersection Observers for animation logic. No external libraries or heavy frameworks were used to prioritize load times and performance.

## Setup & Running Locally 🚀

Because this project is built entirely static with Vanilla HTML/CSS/JS, no build steps or heavy dependencies are required. 

1. Clone or download the repository.
   ```bash
   git clone https://github.com/kido19/FUTURE_FS_03
   ```
2. Open the project folder.
3. You can either:
   - Simply double-click the `index.html` file to instantly open the website in your default web browser.
   - Or, for a better development experience, run it using a local live server extension (like VS Code's "Live Server" or by running `npx http-server ./` in your terminal).
